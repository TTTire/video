# 像素呼吸 — 视频选题工作区

这是「像素呼吸」YouTube/抖音频道的视频制作工作区。频道定位：10分钟认知提升类视频，用经济学 × AI认知的交叉视角讲故事。

## 脚本制作规范

**所有脚本必须严格遵循 `模板/标准脚本模板.md`**。每次写脚本前，先读取该模板文件。

### 核心规则

1. **同时产出两个文件**：分镜脚本 + 口播逐字稿（放在同一个选题文件夹下）
2. **分镜脚本格式**：必须使用标准表格（`| 序号 | 类型 | 脚本旁白 | 视觉元素 | 时长 |`）
3. **口播逐字稿**：中文 ≥ 2800字，一行一句，用 `## 第X章：章节标题` 分隔
4. **四章结构**：每章标题格式 `### 第X章：[标题]（时间段）` + `**本章核心：**`
5. **基本信息**：开头必须有标准表格（视频主题、核心逻辑、视频时长、总镜头数）
6. **自检清单**：分镜脚本末尾必须附10项自检清单
7. **视频 Prompt 后缀**：`Cinematic lighting, hyper-realistic, dark moody color palette, 8k resolution`

### 写作风格

- 悬念/反常识开头，禁止直接讲背景
- 每章至少1次身份代入句（"你……"）
- 每章开头有注意力重置句
- 数据必须有对比锚点
- 至少3句可独立传播的金句
- 口语化短句，一句话不超过25字
- 禁止书面语（"综上所述"）和AI腔调（"深度赋能"）

### 文件夹结构

```
进行中/选题名/
├── 选题名-分镜脚本.md
├── 选题名-口播逐字稿.md
├── 选题名-口播逐字稿-剪映版.md    # 纯文本，一行一句，无标记
├── 选题名-双语字幕.srt           # 录制后生成
└── 选题名-发布清单.md
```
