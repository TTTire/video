# 像素呼吸 — 视频选题工作区

这是「像素呼吸」YouTube/抖音频道的视频制作工作区。频道定位：10分钟认知提升类视频，用经济学 × AI认知的交叉视角讲故事。

## 脚本制作规范

### 核心规则

1. **产出文件**：口播逐字稿 + 发布清单（放在同一个选题文件夹下）
2. **口播逐字稿**：中文 ≥ 2800字，一行一句，用 `## 第X章：章节标题` 分隔
3. **四章结构**：每章标题格式 `### 第X章：[标题]（时间段）` + `**本章核心：**`
4. **视频 Prompt 后缀**：`Cinematic lighting, hyper-realistic, dark moody color palette, 8k resolution`

### 写作风格

- 悬念/反常识开头，禁止直接讲背景
- 每章至少1次身份代入句（"你……"）
- 每章开头有注意力重置句
- 数据必须有对比锚点
- 至少3句可独立传播的金句
- 口语化短句，一句话不超过25字
- 禁止书面语（"综上所述"）和AI腔调（"深度赋能"）

### 文件夹结构

```
进行中/选题名/
├── 选题名-口播逐字稿.md
├── 选题名-口播逐字稿-剪映版.md    # 纯文本，一行一句，无标记
├── 选题名-双语字幕.srt           # 录制后生成
└── 选题名-发布清单.md
```

### 发布清单规范

每次视频完成后必须产出发布清单，**参考范例：`进行中/沉没成本/沉没成本-发布清单.md`**。

清单必须包含以下板块：

1. **发布前检查**
   - 视频质量（时长、hook、字幕、音量、封面）
2. **发布设置**
   - 平台与账号确认
   - 3个推荐标题（含推荐理由）
   - 标签
   - 发布文案（描述）
   - 发布时间（首选/次选/保底）
3. **发布后**
   - 播放检查、评论互动、数据截图
4. **下一条视频**
   - 按选题库推荐下一期，确认结尾已埋预告钩子

### 双语字幕规范（SRT）

每条字幕必须包含两行文本，格式如下：

```
序号
时间轴
中文原文（去标点）
英文翻译（去标点）
（空行）
```

**翻译要求：**
- 第一行：中文原文，去掉所有标点符号（，。？！、；：等）
- 第二行：英文翻译，自然口语化，去掉句末标点，适合字幕阅读
- 英文保持简洁短句风格，与中文原意对应
- 专有名词保留原名（ChatGPT、Daniel Kahneman 等）
- 百分比、数字等保留原格式

**示例：**
```
1
00:00:00,166 --> 00:00:02,366
你有没有觉得AI特别懂你
Do you ever feel like AI really gets you
```
