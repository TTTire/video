# 视频脚本模板

> 10分钟认知提升类视频 · 完整制作模板
> 每次写脚本必须同时产出两个文件：分镜脚本 + 口播逐字稿

---

## 选题文件夹规范

每个选题一个文件夹，放在「进行中」目录下，完成后移至「已完成」。

```
选题名/
├── 选题名-口播逐字稿.md          # 带章节标记，剪辑时对照用
├── 选题名-口播逐字稿-剪映版.md    # 纯文本一行一句，导入剪映识别字幕
├── 选题名-双语字幕.srt           # 带时间戳，上中文下英文
└── 选题名-发布清单.md            # 标题、标签、发布时间、下一条规划
```

### 各文件说明

| 文件 | 用途 | 格式要求 |
|------|------|----------|
| `口播逐字稿.md` | 带章节标记的正式口播稿，剪辑时对照 | Markdown格式，用 `## 第X章` 标记章节 |
| `口播逐字稿-剪映版.md` | 纯文本版，导入剪映做字幕识别 | 一行一句，无表格/序号/视觉描述/章节标记 |
| `双语字幕.srt` | 发布用的中英双语字幕 | SRT格式，带时间戳，每条字幕上中文下英文 |
| `发布清单.md` | 发布前的检查和规划 | 包含标题候选、标签、文案、发布时间、下一条视频 |

### 字幕工作流

1. 从剪映导出字幕 SRT 文件
2. 交给 Claude 处理为双语格式（上中文下英文，保留时间戳）
3. 得到最终的 `选题名-双语字幕.srt`

### 口播逐字稿硬性要求
- **字数**：中文 ≥ 2800字（TTS转10分钟音频需要约3000字）
- **格式**：一行只写一句话，方便TTS断句
- **章节标记**：用 `## 第X章：章节标题` 分隔，方便剪辑做分段效果
- **禁止内容**：不能有序号、表格、视觉描述、Prompt、时长标注
- **纯口播**：只保留嘴里说出来的话

## 三、写作技巧规则（必须遵循）

### 开头钩子（前3句话决定生死）

**必须用悬念/反常识开场，禁止直接讲历史或背景。**

✅ 正确示范：
```
你有没有发现一个很奇怪的事？
周围所有人都在说没钱了，经济不好了，消费降级了。
但有一样东西，用的人反而越来越多。
```

❌ 错误示范：
```
2008年，全球金融危机，华尔街崩了……
（观众：跟我有什么关系？划走）
```

**钩子公式：**
- 反常识结论 → 制造好奇 → 预告价值
- "你有没有发现……" / "有一件事很奇怪……" / "大部分人都不知道……"
- 前10秒必须让观众觉得"跟我有关"

### 身份代入（让观众觉得在说他）

每章至少1次代入句：
- "你现在打开手机看看……"
- "你回想一下自己的经历……"
- "你有没有过这种时候？"
- "你想想看……"

### 注意力重置（每章开头重新抓一次）

每章开头加一句"重新抓人"的话：
- 第二章："好，现在请你记住这三个密码。"
- 第三章："注意听，这个部分可能会颠覆你的认知。"
- 第四章："好，道理讲完了。最后一个章节，我们回到你自己。"
- 也可以用"但等一下"、"你注意"、"这里有个关键区别"

### 数据锚点（数字要有对比才有感觉）

❌ "ChatGPT有4亿用户"
✅ "ChatGPT周活跃用户突破4亿。4亿是什么概念？全中国上网的人也就10亿出头。"

❌ "省了5800块"
✅ "以前雇一个客服每月6000块，现在用AI工具200块搞定80%，省了5800。"

### 节奏控制

- **第一章**：快节奏，密集信息，3秒一个新信息点
- **第二章**：放慢，讲清原理，用类比让人听懂
- **第三章**：加速，密集论证，制造"原来如此"的感觉
- **第四章**：回归平缓，给出可执行的建议，金句收尾

### 语言风格

- 口语化、接地气，像跟朋友聊天
- 短句为主，一句话不超过25个字
- 禁止书面语："值得注意的是"、"综上所述"、"不言而喻"
- 禁止AI腔调："显著提升"、"深度赋能"、"全面覆盖"
- 多用比喻："就像买口红一样，先试色"
- 多用反问："你有没有想过，为什么？"

### 金句设计（至少3句，适合做封面/标题）

每个脚本必须包含至少3句可以独立传播的金句：
- 放在章节结尾或过渡处
- 要求：短、有力、反常识或有画面感
- 示范："口红让你好看，AI让你值钱"

## 四、分镜脚本格式

| 序号 | 类型 | 脚本旁白 | 视觉元素 | 时长 |
|------|------|----------|----------|------|
| 01 | 🎬 视频 | [旁白内容] | Prompt: [...] Cinematic lighting, hyper-realistic, dark moody color palette, 8k resolution | 10-15s |
| 02 | 🖼️ 图片 | [旁白内容] | [图片描述] | 4-5s |

### 镜头类型说明
- 🎬 **视频镜头**：关键转折点、情绪高潮、重要概念（10-15秒）
- 🖼️ **图片展示**：辅助说明、数据展示、概念图（4-5秒）

### 配比建议（10分钟）
- 视频镜头：10-12个（约140-180秒）
- 图片展示：80-93个（约400-460秒）

### 视频生成 Prompt 统一后缀
`Cinematic lighting, hyper-realistic, dark moody color palette, 8k resolution`
