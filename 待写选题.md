### 优先开写建议

- [ ] **#11「一袋盐，为什么能撑起一个帝国？」**（口播+发布清单已生成）
  盐铁专营+垄断利润+国家财政。厨房里最便宜的盐，曾经是古代最硬的印钞机。

- [ ] **#12「一杆秤，为什么能看懂人性和市场？」**（口播+发布清单已生成）
  信任成本+交易秩序+公平感。没有秤的市场，每一笔交易都要靠吵架完成。

- [ ] **#13「一张纸，为什么改变了中国人的命运？」**
  知识复制成本+传播效率+AI生产力。纸降低复制成本，印刷术降低传播成本，AI降低表达成本。

- [ ] **#14「一杯茶，讲透中国人的社交货币」**
  礼物经济+面子账户+关系定价。茶不是饮料，是中国人最温和的一种交易媒介。

- [ ] **#21「陶渊明为什么不是躺平，而是主动降本？」**
  机会成本+固定成本+生活方式选择。他不是不努力，而是拒绝把人生卖给一个不划算的系统。

- [ ] **#1「一部手机为什么越来越贵？」**

## 方向B：AI时代×经济学（差异化主线）

- [ ] **#2「DeepSeek 1块钱100万字：免费才是最贵的」**
  掠夺性定价+平台锁定+边际成本趋零。为什么V4发布当天别家股价暴跌？

- [ ] **#4「AI替你工作8小时，你的工资会涨还是跌？」**
  卢德谬误+劳动力替代+比较优势。历史上每次技术革命的就业真相。

- [ ] **#8「你的脸值多少钱？AI定价时代来了」**
  算法定价+价格歧视+数据资产。同一家酒店，iPhone用户看到的价格比安卓贵30%。

## 方向C：书籍/报告拆解（知识密度高）

- [ ] **#3「《穷查理宝典》：芒格说普通人最大的错误是什么？」**
  多元思维模型+逆向思维+能力圈。一本书浓缩巴菲特搭档60年的投资智慧。

- [ ] **#7「华尔街最新AI报告：2026年哪些人会被淘汰？」**
  拆解真实报告数据，白领替代率+幽灵GDP+税基萎缩。

- [ ] **#9「《纳瓦尔宝典》：不靠上班实现财富自由的底层逻辑」**
  杠杆+专属知识+判断力。硅谷最火的一本财富哲学书。

## 方向D：人物传记×认知提升

- [ ] **#5「梁文锋：DeepSeek背后的中国AI狂人」**
  量化基金起家→做AI→价格打到地板→主动认输说落后3-6个月。反常规的商业决策逻辑。

- [ ] **#10「黄仁勋：一个炒菜工如何成为全球第一CEO？」**
