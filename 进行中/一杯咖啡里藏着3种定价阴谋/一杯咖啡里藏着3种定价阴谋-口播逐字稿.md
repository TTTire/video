# 一杯咖啡里藏着3种定价阴谋 — 10分钟口播稿

我先问你一件事。

你走进星巴克，店员问你——中杯还是大杯？

你想都没想，回了一句"大杯"。

恭喜你——你刚刚踩进了一个布置了40年的陷阱。

我先告诉你一个冷知识。

星巴克的"中杯"，其实是它菜单上最小的那一杯。

它叫Tall——12盎司。

中杯之上，还有大杯Grande 16盎司、超大杯Venti 20盎司。

也就是说——你以为自己在选"中等"，其实你在选"小"。

你以为自己加了几块钱"升一档"，其实你只是回到了正常杯型的位置。

这个菜单，是星巴克在1986年定下来的。

40年了，全球数十亿人每天踩进去，还以为自己捡了便宜。

真相是——价格不是给你看的，是商家算给自己看的。

今天我用10分钟，把一杯咖啡里藏着的3种定价阴谋讲给你听。

听完之后，你下一次点咖啡，至少能省3块钱。

更重要的是——你会发现这3个套路，不只在咖啡店。

奶茶、电影票、视频会员、培训课——你全在踩。

## 第一章：诱饵效应——你以为的"中"，其实是商家的"小"

**本章核心：** 中间那个选项，从来不是给你"对比"的，它是商家算好让你买的。

你回想一下，你上一次站在柜台前，是不是这种感觉——

"小杯太抠了，超大杯喝不完，中等的最合适。"

注意听，这个想法从一开始就是商家植进你脑子的。

1986年，霍华德·舒尔茨把星巴克变成全球连锁。

他做的第一件事，就是设计了一个三杯型菜单。

12盎司、16盎司、20盎司——价格分别是27、32、35。

你看这个差距。

12到16，多了4盎司，加5块。

16到20，多了4盎司，只加3块。

你心里的小算盘立马响起来——再加3块多4盎司，太划算了。

于是70%的人选了大杯。

但他们本来想点的，其实是中杯。

这就是诱饵效应。

它是2008年由行为经济学家丹·艾瑞里提出的。

简单说——商家会在你的选项里，故意塞一个看起来不划算的"诱饵"。

让你对比之下，觉得另一个选项特别值。

但你不知道的是——那个看起来值的选项，从一开始就是商家想让你买的。

我再给你一个更狠的实验。

《经济学人》杂志做过这样一个测试。

订阅方案——

A 网络版 59美元。

B 印刷版 125美元。

C 印刷加网络版 125美元。

你看到这个，是不是觉得——B就是个废物，跟C一样的价格只有一半内容。

那当然选C啊，网络是白送的。

实验结果——84%的人选C。

然后他们删掉B这个"废物选项"，再给另一组人看。

只有32%的人选C，68%的人选了便宜的A。

你看出来了吗？

B这个选项根本没人买。

但它的存在，让C的销量提升了2.6倍。

B不是给你买的，是给你看的。

它是个诱饵。

星巴克的Venti、电影院的"超大桶爆米花"、奶茶店的"豪华套餐"——全都是这个角色。

它们存在的唯一目的，就是让你旁边那个"中等选项"显得合理。

下一次你站在柜台前，请你提醒自己——

中间那个选项，从来不是给你对比的。

它是商家算好让你买的。

## 第二章：心理账户——为什么40块咖啡你愿意，10块矿泉水你不愿意

**本章核心：** 商家不是在卖咖啡，是在卖一种感觉，你掏的是"感觉账户"里的钱。

我先问你一个反差。

你花40块买一杯星巴克，眼都不眨。

但你去便利店，看到一瓶依云矿泉水卖10块——你立刻觉得贵。

为什么？

我们认真算一下成本。

咖啡的成本——咖啡豆加水加人工加店租——一杯大概5块钱。

矿泉水的成本——水加瓶子加物流——大概1块钱。

咖啡卖你40，溢价35。

矿泉水卖你10，溢价9。

哪个更"宰你"？

显然是咖啡。

但你的心却觉得咖啡正常、矿泉水离谱。

这背后的原因，2017年诺贝尔经济学奖得主理查德·泰勒研究了一辈子。

他发现——人不是把所有钱放在一个钱包里。

人脑里有很多个"心理账户"。

"咖啡账户"——一杯小确幸，30到50块都正常。

"日常水账户"——一瓶水，超过5块就贵。

"奢侈品账户"——一只包2万也能接受。

"工资账户"——同事比你多1000块就睡不着。

这些账户互相独立，一笔钱进了哪个账户，决定了你愿意付多少。

商家最厉害的招——是把你的消费"塞进"一个高接受度的账户。

星巴克怎么做的？

它不卖你"咖啡因"。

咖啡因加糖精，5毛钱一杯。

它卖你的是"上班路上的小确幸"。

是"我在好好对自己"。

是"我有一段独处的时光"。

是"我配得上"的仪式感。

这些感觉，你愿意花40块。

但同样5毛钱成本的咖啡因，如果店员对你说——"这是浓缩咖啡因水，3块钱一杯"，你会觉得贵。

商家卖的是感觉，你以为你在喝咖啡。

你回想一下——你过去一年，在"小确幸"账户里花了多少钱？

下午茶、奶茶、小蛋糕、限定咖啡、网红甜品——加起来可能比你买课的钱还多。

但你记得这些钱花了多少吗？

不记得。

因为它们没进"重要消费账户"，进了"对自己好一点"账户。

这个账户里的钱——花得最多，最不肉疼，最难察觉。

记住这一句——你掏的从来不是钱，是感觉。

## 第三章：获客成本——9.9不是赚到，是商家给你下的押金

**本章核心：** 你以为9.9是个赚到，商家以为9.9是个划算的客户押金。

回到主线，讲第三个套路。

它叫获客成本。

讲之前先问你——

瑞幸9.9一杯，星巴克32一杯，差23块。

瑞幸是不是亏死了？

错。

瑞幸2024年净利润——超过30亿人民币。

它怎么做到的？

我给你拆开账本。

第一部分——咖啡豆加牛奶加杯子加小料，成本3到4块。

第二部分——店面租金加设备加水电，平摊到每一杯1到2块。

第三部分——人工，平摊到每一杯1到2块。

加起来，一杯瑞幸的真实成本——5到8块。

卖你9.9——它每一杯还能赚1到4块。

而星巴克卖32——大几块成本，剩下二十几块全是利润加品牌加体验。

但瑞幸为什么死磕"9.9引流"？

因为它在算的是另一笔账——叫客户终身价值，英文缩写LTV。

它知道——

你今天因为9.9来一次。

下次它推送你19.9的"新品打卡"。

下下次推送你"会员充50送50"。

一个月后你已经是它的"日常用户"，平均每周花40块。

一年下来，你给它贡献2000块。

5年下来，1万块。

这一万块，就是你的LTV。

我身边一个朋友，3年前因为"9.9新人券"第一次喝瑞幸。

现在他每天上班顺手一杯。

我让他打开账户算了一下——

3年下来，他在瑞幸花了8400块。

他看到数字的时候愣了10秒。

他说："我一直以为我喝的是9.9。"

他确实喝的是9.9。

但他每天都喝。

获客成本的真相是——

商家愿意花100块"买"一个客户进来。

只要这个客户未来5年能给它贡献1000块，它就赚翻了。

所以你不是在喝便宜咖啡。

你是商家用低价"买"来的一个未来5年的钱包。

你以为9.9是个赚到。

商家以为9.9是个划算的"客户押金"。

谁的视角更长远？

请你诚实回答。

这个套路，瑞幸在用、拼多多在用、滴滴在用、所有视频网站的"首月1元"都在用。

你贪的是1块钱，他们要的是5年。

## 第四章：3个反操纵的方法+一句话收尾

**本章核心：** 表面上是商家定价，本质上是你在为自己的认知漏洞买单。

讲了这么多套路，怎么破？

我给你3个方法，下次进咖啡店、奶茶店、商场——拿出来用。

第一个方法，叫「跳过中间选项」。

不管商家给你3个还是5个选项，你都从最便宜那个开始问——这个能不能满足我的需求？

能，就买最便宜的。

不能，再考虑下一档。

不要从"中间"开始想。

中间是商家设计给你的诱饵。

你从最便宜那一档反推，你就走出了诱饵效应。

第二个方法，叫「记一笔小账」。

每个月在备忘录里写一行——

我这个月在"小确幸"账户里花了多少钱？

奶茶、咖啡、甜品、零食、订阅会员——加起来。

数字写出来，比你想象中可怕。

300？500？1000？

写3个月——你自然就会节制。

不是让你不花，是让这个账户从"无感"变成"看得见"。

第三个方法，叫「LTV反向算」。

每次商家给你"超低价首单"，问自己一句——

它收的是9.9，它打算从我身上长期赚多少？

你想清楚这个问题——

那些"会员充50送50""首单9.9""新人专享"，立刻没那么有吸引力了。

你看穿了它的游戏，你就不再是它要的那个客户。

我给你模拟一个场景。

你下班路过星巴克，店员问你——中杯还是大杯。

旧的你，脱口而出"大杯"，付32块。

新的你——

第一步，问自己：今天我真的需要那么多咖啡因吗？不需要。

第二步，问店员："我要小杯，请问最小是哪一杯？"店员一愣——"是Tall，12盎司，27块。"

第三步，付27块走人。

省5块。

听起来不多。

但你算算——每周3杯，一年780块，10年7800块。

更重要的不是这7800块。

是你看穿了这个游戏的那一刻，你的"消费理性肌肉"被锻炼了。

下一次面对奶茶店、电影院、会员订阅、培训班——同样的方法你都能用。

10年下来，你会发现自己比身边人多攒了几万块，而且生活质量并没有下降。

因为你只是不再为商家的"诱饵"买单。

其实关于一杯咖啡，最核心的真相只有一句——

价格不是给你的，是商家算给自己看的。

你看到的"中杯"，是它想让你买"大杯"的钓饵。

你看到的"40块小确幸"，是它想让你忽略真实成本的话术。

你看到的"9.9引流"，是它想让你留下来花一万块的押金。

你能识破多少层——决定了你这一辈子省下的钱、攒下的钱、最后能选择的人生。

最后送你一句话——

商家最怕的不是你嫌贵。

是你算账。
