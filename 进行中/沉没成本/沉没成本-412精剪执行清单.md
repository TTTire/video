## 精剪流程（预计3-4小时）

### 第一步：素材整理（30分钟）
- [ ] 把TTS音频+录制素材导入剪辑软件
- [ ] 按4个章节分段，命名好
- [ ] 听一遍，标记需要调整的部分
- [ ] 删除明显失误片段

### 第二步：粗剪（45分钟）
- [ ] 每章保留最好的一条
- [ ] 去除口误、停顿、"嗯啊"
- [ ] 拼接4章，检查整体节奏
- [ ] 总时长控制在 9-11 分钟

### 第三步：精剪 + 素材叠加（90分钟）
- [ ] 第一章（0:00-3:20）：加入订阅列表截图、算账动画、俄亥俄实验柱状图、自助餐场景、电影院场景
- [ ] 第二章（3:20-5:50）：加入AI工具logo墙、年付算账截图、三层机制动画、自由职业者故事配图
- [ ] 第三章（5:50-8:00）：加入股票K线图、A股数据图、感情/工作/培训场景、四场景并排画面
- [ ] 第四章（8:00-10:00）：加入三个方法文字卡、模拟场景、行动号召画面、金句大字卡
- [ ] 关键数字出现时加文字卡片（3亿人、1800块、700块、10%）
- [ ] 加入章节过渡（简单的文字卡or淡入淡出）

### 第四步：细节打磨（30分钟）
- [ ] 背景音乐（4段BGM对应4章，参考截图清单中的BGM建议）
- [ ] 音量平衡：人声为主，BGM压到20%-30%
- [ ] 片头3秒hook检查：前3秒能不能抓住人？
- [ ] 片尾：加关注引导 + 下期预告
- [ ] 字幕检查（自动生成后逐段校对）
- [ ] 金句处字幕加粗或变色

### 第五步：导出 + 封面（15分钟）
- [ ] 导出 1080p
- [ ] 做封面图：大字标题 + 对比感（如"花了1000块只用了3次"）
- [ ] 准备发布文案和标签
